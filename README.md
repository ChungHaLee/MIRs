# MIRS (Music Info Retrievals)

Python 버전은 라이브러리 의존성 때문에 3.8 을 사용하고 있습니다. <br>
requirements.txt 에 있는 외부 라이브러리들을 모두 설치해주세요.


## 라이브러리 설치

```
$ pip install -r requirements.txt
```


## 파일 구조

```bash
├── static
│   └──  data ├── melondata_edited.csv
│   
├── retrieval.py
│
└── requirements.txt
```
retrieval.py 을 실행하면 됩니다!


## 실행

```
$ python retrieval.py
```
